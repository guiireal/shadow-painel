## Instalação

<br />

### 💻 Termux

```sh
pkg upgrade && pkg update -y

pkg install nodejs

npm install

```

<br />

### 💻 Debian

```sh
apt upgrade && apt update -y

apt install nodejs

npm install

```

<br />

## Executar o Painel

```sh
npm start
```
